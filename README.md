# whiteBoard

功能列表：

- 改变线条颜色
- 改变线条宽度
- 画笔
- 直线
- 画矩形
- 画圆
- 填写文本
- 橡皮擦
- 撤销上一步
- 清空画板
- 设置画板背景图片
- 访问地址后跟上查询参数 ?id=roomId 可进入自定义房间免受其他用户干扰

# 启动步骤

```
git clone https://github.com/Qbian61/whiteBoard.git

cd whiteBoard

npm install

node server.js
```

启动成功后访问 ```http://127.0.0.1:8082``` 即可。
